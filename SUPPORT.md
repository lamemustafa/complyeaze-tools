# Support

ComplyEaze Tools is a free, open-source, browser-local tool collection.

## Public Support

Use GitHub issues for:

- documentation gaps;
- synthetic reproduction cases;
- accessibility bugs;
- unsupported input examples that do not include taxpayer data;
- requests for new local-first tools.

## Private Reports

Send security and privacy issues to security@complyeaze.com. Do not open public
issues with taxpayer identifiers, portal screenshots, credentials, cookies,
OTPs, CAPTCHA answers, local paths, notices, ledgers, or document contents.

## Product Boundary

The free tools create review drafts. Axal is the path for recurring
reconciliation, client workflows, credential handling, document vaults, task
assignment, and audit trails.
